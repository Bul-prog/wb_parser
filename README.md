# WB Parser 🧥

Парсер товаров с Wildberries по поисковому запросу.

## 🚀 Возможности

- Сбор карточек товаров по поисковому запросу
- Парсинг:
  - Названия
  - Цены
  - Рейтинга
  - Количества отзывов
  - Характеристик
  - Изображений
  - Остатков
- Сохранение результатов в Excel:
  - полный каталог
  - отфильтрованная выборка

---

## 🛠 Технологии
- Python
- Selenium
- Pandas
